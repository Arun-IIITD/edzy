## Canteen Management App

A React-based web application to manage students and snacks in a school canteen. Users can view snacks, add students, track orders, and see detailed student information.

## Features

View Snacks: Browse all available snacks in a responsive grid.

Add Students: Add new students with Name, Age, and Class.

Student Grid: View all students with referral code and total spent.

Delete Students: Remove students easily.

Student Details: View detailed student info and order history.

Order Snacks: Place snack orders for students (via modal).

Responsive Design: Works on desktop and mobile screen

## Tech Stack
Frontend: React, React Router

State Management: React Context API

Styling: CSS Grid, Flexbox

Data Persistence: LocalStorage

Components: Modular React components (SnackCard, OrderModal, etc.)
